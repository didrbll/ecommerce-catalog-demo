# E-commerce Catalog Page

Этот проект представляет собой адаптивную веб-страницу каталога товаров с функционалом фильтрации, динамической подгрузкой контента и модальными окнами. Проект написан с использованием чистого JavaScript (Vanilla JS) и стилизован с помощью SCSS по методологии BEM.

## 🚀 Особенности

*   **Динамический рендеринг товаров:** Товары отображаются на странице с использованием HTML-шаблонов (`<template>`).
*   **Фильтрация:** Возможность фильтровать товары по категориям (Personal Audio, Hi-Fi Audio, Microphones и т.д.).
*   **Пагинация "Load More":** Кнопка для постепенной подгрузки дополнительных товаров.
*   **Модальные окна:** Реализация диалоговых окон с использованием нативного тега `<dialog>`.
*   **Адаптивный дизайн:** Корректное отображение на мобильных устройствах и десктопах (скрытое бургер-меню, адаптивная сетка товаров).
*   **UI элементы:** Бегущая строка (Marquee), выпадающие меню (Dropdown), иконки от [Remix Icon](https://remixicon.com/).

## 🛠 Технологии

*   **HTML5:** Семантическая разметка, использование тега `<template>` и нативного `<dialog>`.
*   **SCSS (Sass):** Препроцессор для стилей. Архитектура стилей разбита на модули (blocks, helpers, globals, variables) с использованием методологии **BEM**.
*   **JavaScript (ES6+):** Модульная архитектура. Отдельные файлы для логики фильтрации, рендеринга, работы с модальными окнами и дропдаунами.
*   **Шрифты:** Heebo (переменный шрифт).

## 📁 Структура проекта

```text
├── images/                 # Изображения товаров
├── scripts/                # JavaScript логика
│   ├── modules/            # JS модули (dropdown, filter, modal, рендеринг и т.д.)
│   ├── main.js             # Точка входа JS
│   └── productsData.js     # Моковые данные о товарах
├── styles/                 # SCSS стили
│   ├── blocks/             # BEM блоки (компоненты)
│   ├── helpers/            # Глобальные переменные, миксины, утилиты
│   └── style.scss          # Главный файл стилей
├── index.html              # Главная страница разметки
└── logo.svg                # Логотип проекта
```
